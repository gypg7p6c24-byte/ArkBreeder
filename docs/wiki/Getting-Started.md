# Getting Started

## Requirements
- Python 3.12+
- Linux desktop environment with Qt support (tested on Ubuntu)

## Run locally
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e .
arkbreedingtool
```

## First setup
1. Open **Settings**.
2. Check the export watch folder.
3. Import `GameUserSettings.ini` and `Game.ini`.
4. Import or generate creatures from ARK exports.
