# Arc-Breading-Tool Wiki

Arc-Breading-Tool helps you manage ARK creature exports, compute stat points, and plan breeding chains.

Technical command/module names remain `arkbreedingtool`.

Developed with OpenAI Codex.

- [Breeding Workflow](Breeding-Workflow)

## Core ideas
- Import Dino export `.ini` files from your ARK `DinoExports` folder.
- Keep server settings in sync by importing `GameUserSettings.ini` and `Game.ini`.
- Use Breeding overview for quick actions, then open a detailed plan per species.
