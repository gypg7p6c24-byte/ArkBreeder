# Settings and Multipliers

## INI import
Use:
- `GameUserSettings.ini`
- `Game.ini`

These values feed the stat solver (wild/tamed multipliers, imprint scale, and level constraints).

## Manual overrides
If INI files are unavailable or incomplete, you can set:
- Max wild level cap
- OverrideOfficialDifficulty
- DifficultyOffset
- BabyImprintingStatScale

Manual overrides are stored and reused on next launch.
